# Flet Mouseframe Organizer

A Flet application designed to organize and manage data from Mouseframe.pt.

## Features

- Import data from Mouseframe.pt
- Organize and categorize information
- User-friendly interface built with Flet

## Installation

```bash
pip install flet
```

## Usage

```bash
python main.py
```

## Requirements

- Python 3.9+
- Flet
